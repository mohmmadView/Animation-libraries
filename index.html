installing Animate.css

Download animate.css framework from project github

https://github.com/animate-css/animate.css

or

npm install animate.css --save

Linking the animate.css or animate.min.css file in the main index of the project

or

<link rel="stylesheet" href=" https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css">

Usage

<h1 class="animate__animated animate__wobble"> Animate.css </h1>
